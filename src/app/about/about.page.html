Sobre

Apresentação:

Este aplicativo foi desenvolvido como parte da disciplina de Desenvolvimento de Aplicativos Mobile. O objetivo principal é aplicar os conceitos de interfaces híbridas utilizando o framework Ionic. A estrutura do projeto baseia-se no Angular, aproveitando do TypeScript, tipado e de fácil manutenção. Durante o desenvolvimento, foram explorados componentes nativos, navegação entre telas (Router), estilização customizada com SCSS e manipulação de eventos de interface
